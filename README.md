# バッケットネコ研究所
> 創作キャラクターの設定やイラストを閲覧できるサイト
## 概要
「バケットネコ研究所」は一時創作で作成した「バケットネコ」をはじめとした「自作キャラクター」の情報を管理するポートフォリオ的サイトです。
## 主なキャラクター
- 🪣😺 バケットネコ
## 技術スタック
* **Framework**: Next.js 16 (App Router)
* **Language**: TypeScript
* **Styling**: Tailwind CSS
## 起動方法
### 前提条件
- Node.js `v23.7.0`以上
### 前提条件
1.  **リポジトリをクローン**
```bash
    git clone [https://github.com/CatNighp/bucket-cat-lab.git](https://github.com/CatNighp/bucket-cat-lab.git)
    cd bucket-cat-lab
    ```

2.  **依存関係のインストール**
```bash
    npm install
    # または pnpm i, yarn install
    ```

3.  **環境変数の準備**
```bash
    cp .env.example .env
    ```

4.  **開発サーバーの起動**
```bash
    npm run dev
    ```
自動的にブラウザが開かない場合は、`http://localhost:3000` にアクセスしてください。
